
# s2w-js 
Library based on raw JavaScript 

## Install 
``` shell 
npm install s2w-js -S 
``` 

### On demand import 
``` javascript 
import { 
  isArr
  isBoo
  isEnum
  isFun
  isJQ
  isNan
  isNull
  isNum
  isObj
  isStr
  isUnd
  except
  plus
  reduce
  ride
  browser
  color
  forEach
  initial
  random 
} from 's2w-js' 
// or 
const { 
  isArr
  isBoo
  isEnum
  isFun
  isJQ
  isNan
  isNull
  isNum
  isObj
  isStr
  isUnd
  except
  plus
  reduce
  ride
  browser
  color
  forEach
  initial
  random 
} = require('s2w-js') 

// use 
isArr()
isBoo()
isEnum()
isFun()
isJQ()
isNan()
isNull()
isNum()
isObj()
isStr()
isUnd()
except()
plus()
reduce()
ride()
browser()
color()
forEach()
initial()
random() 
``` 

### Complete introduction 
``` javascript 
import s2wJS from 's2w-js' 
// or 
const s2wJS = require('s2w-js') 

// use 
s2wJS.$XXX() 
``` 

## CDN 
``` html 
<script src="https://cdn.jsdelivr.net/npm/s2w-js@1.0.9/dist/s2w-js.min.js"></script> 
``` 
``` javascript 
s2wJS.$XXX() 
// or 
s2w.$XXX() 
```
